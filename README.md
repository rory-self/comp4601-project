# Hardware Accelerated Decompression using Arithmetic Coding
## By Rory Self, Bryan Bong, Ujjwal Uberoi and Sadat Kabir
In this project, we aim to use High-Level RTL Synthesis to accelerate 'Arithmetic Coding' on an Kria KV260 adaptive SoC. Arithmetic Coding is a form of 'entropy coding', a standard lossless data compression technique that minimises the number of bits required to represent data. This is achieved by effectively substituting the most frequent patterns of binary strings with shorter mapped alternatives.
