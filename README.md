# Hardware Accelerated Compression using Arithmetic Coding
## By Rory Self, Bryan Bong, Ujjwal Uberoi and Sadat Kabir
In this project, we aim to use High-Level RTL Synthesis to accelerate 'Arithmetic Coding' on a Kria KV260 adaptive SoC. Arithmetic coding is a form of 'entropy coding', a standard lossless data compression technique that minimises the number of bits required to represent data. This is achieved by calculating a probability model of symbols in a dataset, and systematically using corresponding interval subdivisions to collapse the bitstring into a single number and smaller binary length.
